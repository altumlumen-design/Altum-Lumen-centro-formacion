GUÍA DE CONSULTA PÚBLICA

¿Cómo usar el aplicativo?

  1. Ingrese el número de documento o el nombre completo del participante.
  2. Presione Buscar.
  3. El sistema mostrará los registros válidos, anulados o reemplazados que coincidan con la búsqueda.
  4. Use Imprimir si desea generar una copia de la consulta.
  5. Para una nueva búsqueda, pulse Limpiar.

Nota: este aplicativo consulta el archivo de datos configurado dentro del proyecto. Para actualizar registros, reemplace su archivo data/certificados.json o ajuste la ruta en el script.
