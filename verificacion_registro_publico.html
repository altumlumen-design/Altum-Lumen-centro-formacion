¿Cómo usar el aplicativo?

  1. Ingrese solo un criterio: DNI, nombre completo o código de emisión.
  2. Escriba el código de validación visual mostrado en pantalla.
  3. Presione Buscar.
  4. Si existen varios documentos para el mismo criterio, el sistema mostrará todos.
  5. Use Imprimir si desea generar una copia de la consulta.
